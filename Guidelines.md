## Overview

### What You're Building

You will design, build, and deploy a fully functional AI agent — end to end.

This is not a toy demo. You will:
- Define a real problem  
- Architect the solution  
- Implement it using production tools  
- Deploy it live on the internet  

Over the next 2–3 classes, implementation steps will be guided. Your task is to extend that into a complete, working system.

## Team Structure

### Groups of Two (Role-Based)

Each group consists of two members.

#### Role A — Architect & Integrator
- Problem definition  
- Task decomposition  
- Architecture design  
- API and tool integrations  
- Owns system design documentation  

#### Role B — Builder & Deployer
- Agent implementation  
- LLM-as-Judge integration  
- UI development  
- Deployment  
- Owns application and Loom video  

> Both members must understand the entire system.  
> Evaluation during viva is individual.

---

## Deliverables

### 1. Problem Statement (2 Marks)
- Clearly define:
  - User  
  - Problem  
  - Need  
- Justify why an **agentic approach** is suitable  

---

### 2. Task Decomposition & Specifications (5 Marks)
- Break system into tasks  
- Define:
  - Inputs  
  - Outputs  
  - Decision points  
- Explain how each step is handled  

---

### 3. Architecture Diagram (5 Marks)
- High-level system diagram  
- Must include:
  - Components  
  - Data flow  
  - APIs  
  - Tools  
- Should be understandable to an external viewer  

---

### 4. Implementation (Core)

#### API Connection (4 Marks)
- Use:
  - Gemini API **or**
  - Ollama (local inference)

#### Tool Integration (4 Marks)
- Integrate **Tavily Search**
- Agent must invoke it dynamically  

#### LLM-as-Judge (4 Marks)
- Define evaluation rubric  
- Use LLM to evaluate output quality  

#### Working Agent (10 Marks)
- End-to-end pipeline:
  - Input  
  - Reasoning  
  - Tool usage  
  - Output  

#### Deployment (3 Marks)
- Deploy on:
  - Vercel **or**
  - Railway  
- Must be publicly accessible  

---

### 5. Loom Video (3 Marks)
- Maximum 3 minutes  
- Include:
  - Architecture explanation  
  - Key implementation decisions  
  - Live demo  

---

### 6. Viva (10 Marks)
- 10-minute Google Meet  
- Individual evaluation  
- Topics:
  - System design  
  - Implementation  
  - Technical understanding
